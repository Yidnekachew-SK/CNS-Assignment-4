# CNS-Assignment-4

This assignment demonstrates federated login using Google OAuth 2.0. The demo integrates a Google Sign‑In button into a simple webpage. After successful login, the page displays the name “**Yidnekachew Sitotaw**”. 

### Files  
**index.html** – main page with Google Sign‑In button and layout.  
**script.js** – handles the login callback and displays the name.  
**style.css** – styles the webpage contents.  

### How it works?
Open the deployed site [(GitHub Pages link)](https://yidnekachew-sk.github.io/CNS-Assignment-4/).  
Click the Google Sign‑In button.  
Sign in with a Google account.  
After successful login, the page shows "Yidnekachew Sitotaw".
